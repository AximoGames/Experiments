# AxMarbles

Get the sources:

```
mkdir AximoGames
cd AximoGames
git clone --recursive https://github.com/AximoGames/AxEngine.git
git clone --recursive https://github.com/AximoGames/Experiments.git
```

Build from command line:

```
cd Experiments
dotnet build
```

Run:
```
dotnet run -p <Experiment Name>
```
